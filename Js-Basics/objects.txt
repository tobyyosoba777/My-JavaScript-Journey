Object.fromEntries() =======================================>
============================================================>

The Object.fromEntries() static method transforms a list of key-value pairs into an object
In JavaScript, a static method is a method that belongs to a class itself, rather than to 
individual instances of the class. This means that you can call a static method directly on 
the class name, without needing to create an instance of the class first.
